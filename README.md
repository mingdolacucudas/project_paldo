## 프로젝트명: 팔도어때 


### 프로젝트 설명

기나긴 코로나 시국이 막바지에 이르고, 거리두기가 완화되면서 사람들의 바깥활동이 증가하였습니다. 그동안 즐기지 못했던 축제들을 찾는 사람들도 많아졌습니다.
이러한 소비자들의 니즈에 따라 국내축제를 모아볼수 있고, 예매, 리뷰도 남길수 있는 웹사이트를 제작하였습니다.
